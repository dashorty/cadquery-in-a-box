# cadquery-in-a-box.
Docker compose setup for cadquery-server

## How to use it

checkout repository
```bash
git clone https://github.com/dashorty/cadquery-in-a-box.git
```

start docker compose

```bash
docker compose up -d
```
cadquery-server is running on `http://127.0.0.1:5000`
